💡 Apa itu Playground Scenarios?

Playground scenarios adalah branches yang sudah di-setup spesifik untuk practice konsep tertentu. Setiap scenario adalah independent dan bisa di-reset kapan saja tanpa affect main branch Anda.

Keuntungan: Practice berulang, tidak takut membuat mistake, learn by doing!

Scenario 1

First Commit

⏱️ ~10 menit

Practice membuat file, stage, commit, dan push untuk pertama kalinya tanpa pressure.

Pelajari: git add, git commit, git push, commit workflow
Checkout branch ini:
git checkout scenario-1-first-commit

🔄 Bagaimana Cara Reset Scenario?

Jika Anda membuat mistake atau ingin ulang scenario:

git reset --hard origin/scenario-X-name

Branch akan di-reset ke state awal. Aman untuk di-ulang berulang kali!

📊 Recommended Practice Order

  1. Scenario 1 - First Commit (fundamental)
  2. Scenario 2 - Branching (understand branches)
  3. Scenario 3 - Merge Conflict (problem-solving)
  4. Scenario 4 - Team Collaboration (team dynamics)
  5. Scenario 5 - GitHub Actions (automation)

Pro Tip: Ulang setiap scenario 2-3x sampai feel confident sebelum move ke scenario berikutnya!

Playground adalah tempat untuk experiment tanpa takut! 🚀
