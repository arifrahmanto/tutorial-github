## Why

Banyak developer baru merasa tertinggal dalam memahami Git dan GitHub sebagai workflow kolaborasi, bukan hanya version control tools. Repository ini akan menyediakan pembelajaran project-based yang terstruktur dengan playground interaktif, sehingga developer baru dapat memahami konsep dasar, berkontribusi ke open source, dan menguasai workflow tim secara praktis.

## What Changes

- **8 modul pembelajaran terstruktur** mencakup: pembuatan repository, perintah dasar, manajemen cabang, kolaborasi tim, resolusi konflik, strategi Git, GitHub Pages, dan GitHub Actions/Netlify deployment
- **Playground repository** dengan 5 skenario belajar yang dapat direplikasi ulang
- **4 learning projects** yang progressif: pemula (first repo), tim kecil (kolaborasi), advanced (conflict resolution), OSS-style (fork & contribute)
- **Static site** di GitHub Pages yang visualisasi Git workflows dan tutorials
- **Progress tracking** dengan checklist per modul dan sample project
- **GitHub Actions + Netlify integration** untuk automated deployment dan learning feedback

## Capabilities

### New Capabilities

- `git-fundamentals`: Perintah dasar Git, pembuatan repository, commit workflow
- `branch-management`: Strategi branching, branch naming, checkout dan merge operations
- `team-collaboration`: Pull requests, code review, collaborative workflows di GitHub
- `conflict-resolution`: Merge conflict handling, rebase workflows, resolving strategies
- `git-strategies`: Implementasi Git dalam SDLC (Git Flow, GitHub Flow, trunk-based)
- `github-pages-deployment`: Setup dan deployment static site via gh-pages branch
- `ci-cd-with-actions`: GitHub Actions untuk automation, Netlify integration untuk deployment
- `open-source-contribution`: Fork workflow, upstream tracking, contributing best practices
- `progress-tracking`: Checklist system untuk monitor learning progress per modul
- `playground-scenarios`: Reproducible Git scenarios untuk hands-on practice

## Impact

- **Repository structure**: Akan ada 4 folder utama baru (`docs/`, `playground/`, `sample-projects/`, `github-pages-demo/`)
- **GitHub configuration**: Setup GitHub Pages dengan branch `gh-pages`, workflow untuk Netlify deployment
- **CI/CD**: GitHub Actions workflow untuk build & deploy static site
- **Documentation**: Konten pembelajaran dalam Bahasa Indonesia, README sebagai entry point
- **Target developers**: Developer baru (junior) yang ingin memahami Git/GitHub secara mendalam
