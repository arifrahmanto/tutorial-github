# Playground Scenarios

Welcome to the **Git Learning Playground**! Each scenario is designed for hands-on practice with Git concepts.

---

## 🎮 How to Use Playground

### For Each Scenario:

1. **Clone repository**
   ```bash
   git clone <repo-url>
   cd tutorial-github
   ```

2. **Checkout scenario branch**
   ```bash
   git checkout scenario-1-first-commit
   ```

3. **Follow README instructions** in each scenario folder

4. **Reset if needed** (start over without breaking anything)
   ```bash
   git reset --hard origin/scenario-X-name
   ```

## ✅ Getting Started

Start with **Scenario 1** if new to Git!

```bash
git checkout scenario-1-first-commit
cat README.md  # Follow instructions
```

Each scenario is **independent** - can do in any order after learning prerequisites.

---

## 💡 Tips

- **Stuck?** Check `git status` to understand current state
- **Wrong command?** Use `git reset --hard origin/scenario-X` to start over
- **Confused?** Re-read modul yang corresponding
- **Want to practice again?** Reset branch dan try again
